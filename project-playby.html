Project Overview

This social app connects local players, enabling them to upload, edit, and share custom videos. The video feed mimics short-style video formats like TikTok, with interactive features such as tagging, following, and content engagement.

Key challenges included implementing seamless video recording, editing features within the app, ensuring smooth playback, and integrating offline caching for better performance in low-connectivity areas.

  • Video recording and editing features integrated directly within the app
  • Custom video player for a smooth shorts-style experience
  • Player tagging and following functionality
  • Offline caching for video feed and uploads
  • Optimized image and video handling for better app performance
  • Clean architecture for scalable and maintainable app growth

Technical Implementation

Architecture

The app is built using Clean Architecture principles with Provider for state management and dependency injection. This structure promotes scalability, testability, and a clear separation of concerns. The project is organized into the following layers:

  • Presentation Layer: Contains UI components and their corresponding providers, handling user interaction and state updates.
  • Domain Layer: Encapsulates core business logic, use cases, and entity definitions independent of Flutter or external libraries.
  • Data Layer: Implements repositories and data sources (local and remote) to manage data flow and handle API interactions.

Dependency injection is managed using get_it, enabling easy injection of use cases, services, and repositories into widgets and logic components.

Performance Optimization

Several optimizations have been implemented to ensure a responsive and efficient user experience:

  • Image caching using cached_network_image for smoother image loading
  • Lazy loading and pagination for property listings
  • Minimized widget rebuilds using const constructors and selective Consumer usage
  • Background isolates for intensive operations (e.g., image processing)
